# Heart Rate Zone Monitor

A low-cost wearable that monitors heart rate in real time and provides immediate feedback through LEDs, vibration, and an LCD display. Built with Arduino as a wearable prototype for heart rate zone training.

**Demo:** https://drive.google.com/file/d/10NzonvZ3ZtQ_E06AMcp3S1LnRGKjvqoF/view?usp=sharing

---

## Overview

The device allows users to enter their age and resting heart rate, continuously measures BPM using an optical heart rate sensor, and calculates personalized heart rate zones using the Karvonen formula. As the user's heart rate changes, the wearable updates the display, changes LED color to indicate the current zone, and vibrates whenever a new zone is entered.

---

## Features

- Real-time heart rate monitoring
- Personalized heart rate zone calculations
- Five-color LED zone indicator
- Vibration alerts for zone transitions
- LCD interface for user input and live BPM display
- Built using low-cost, off-the-shelf components

---

## Hardware

- Arduino Uno
- Optical heart rate sensor
- RGB LED
- LCD shield
- Vibration motor
- 3D printed enclosure

---

## Results

- Heart rate accuracy within **±2.5 BPM** compared to an Apple Watch
- Built for **under $100**
- Fully integrated wearable prototype
